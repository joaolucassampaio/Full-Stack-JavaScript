Quiz 4

1/5. Sobre as funções no javascript, marque a alternativa incorreta:

a\ Uma função só pode ser chamada no código após a sua inicialização, não é possível referenciar uma função acima da sua declaração.

b\ Tanto os parâmetros quanto o retorno de uma função são opcionais.

c\ Para criar uma função utilizamos a palavra function, definimos os parâmetros entre parêntesis e utilizamos chaves para agrupar o conteúdo da função.

d\ Uma função pode ser manipulada no código como um outro objeto qualquer, sendo passada como parâmetro, armazenada em uma variável ou chave de objeto, desde que seja utilizado o nome da função sem os parêntesis.

e\ Uma função pode ser vista como uma forma de armazenar um procedimento e nomeá-lo, assim é possível chamá-lo quando e onde necessário no código.

Resposta: "A".


2/5. Qual das alternativas abaixo representa a forma correta de se criar uma função anônima:

a\ const funcao = function() {//...}

b\ anonymous function funcao() {//...}

c\ function() {//...}

d\ const funcao = anonymous function() {//...}

e\ function funcao() {//...}

Resposta: "A".


3/5. O que é uma high-order function?

a\ É uma função que não possui nome.

b\ É uma função que chama ela mesma.

c\ É uma função que recebe uma outra função como parâmetro.

d\ É uma função salva dentro de uma variável.

e\ É uma função que é passada como parâmetro para outra função.

Resposta: "C".


4/5. Em arrays do javascript, o que faz a função “map”?

a\ Apenas itera sobre cada elemento de um array.

b\ Itera sobre cada elemento de um array e cria um novo array filtrando elementos dele.

c\ É utilizado para ordenar arrays.

d\ Itera sobre cada elemento de um array e cria um novo array baseado nele.

e\ É utilizado para transformar arrays em objetos.

Resposta: "D".


5/5. O que é uma função recursiva?

a\ É uma função que chama ela mesma.

b\ É uma função que é passada como parâmetro para outra função.

c\ É uma função salva dentro de uma variável.

d\ É uma função que não possui nome.

e\ É uma função que recebe uma outra função como parâmetro.

Resposta: "A".

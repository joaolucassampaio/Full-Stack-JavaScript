Fundamentos Básicos do JavaScript

O que é uma Linguagem de Programação?

É um conjunto de códigos que contém instruções para o computador. Diferente de linguagem de marcação (como HTML, XML, YAML etc) e de linguagem de estilização (como CSS). Alguns exemplos de linguagens de programação: C, C++, Java, C#, PHP, Phyton, Ruby, Javascript, Go, Elixir, Dart e muitas outras.

As linguagens de programação podem ser de alto nível ou de baixo nível. Sendo, as primeiras, mais fáceis de compreender e mais próximas da linguagem humamana. Enquanto, as segundas, são mais próximas do nível de entendimento do hardware.

O que é Javascript?

Javascript é uma linguagem de programação multiparadigma e de alto nível, criada em 1995 por Brendan Eich para ser usada pelo Netscape. Seu nome oficial é ECMAScript. Ela é a principal linguagem de programação usada pelos navegadores web e, junto com HTML e CSS, forma as três principais tecnologias da web. Também pode ser usada fora dos navegadores através de runtimes como o Node.js

Tipos de Dados

Em uma linguagem de programação, dados referem-se a informações que são manipuladas e processadas pelos programas. No javascript (e em outras linguagens de programação) os dados podem ser representados em diferentes formas:

Tipos Primitivos

  1. Number e Bignt
  2. String
  3. Boolean
  4. Null e Undefined

Os primeiros retornam valores numéricos, sendo o Bignt para números muito grandes (quando o valor possui muitas casas decimais, usa-se a letra "n" no final do número, para se evitar erro). O segundo retorna valores de texto. O terceiro retorna valores "True" ou "False". Já os últimos, retornam valores vazios e indefinidos, respectivamente.

Variáveis

Variáveis são elementos fundamentais na programação, servindo como contêineres para armazenar e manipular dados. No Javascript existem três tipos de variáveis, sendo elas: var (variable), let (let it change) e const (constant). A variável var cria um valor que pode ser alterado ao longo do código. A variável let funciona da mesma forma que a var, mas a sua escrita é mais aceita no meio da programação. A variável const cria um valor imutável, ou seja, não pode ser alterado ao longo do código.

É importante, para nomear essas variáveis, dar nomes significativos e padronizados. Para fazer com que um nome de uma variável seja significativo, basta fazê-lo corresponder ao dado que ela for armazenar. Por exemplo: let nome = "Joao". Para fazer com que a digitação das variáveis seja padronizada, seguimos modelos denominados "cases". Sendo os mais famosos: snake e camel. As variáveis escritas conforme o primeiro case são nomeadas da seguinte forma: let nome_do_usuario = "Joao". Para o segundo: let nomeDoUsuario = "Joao".

Operadores

  1. Aritméticos +, -, *, / e %
  2. Agrupamento ( )
  3. Atribuição =, +=, -=, *=, /= e %=
  4. Encadeamento .

Comentários

Os comentários são linhas do código que serão ignoradas pelo interpretador. Servem para deixar anotações e ajudar a entender e documentar o código.

Como usar?

  1. Uma linha //
  2. Várias linhas /*
  3. */

Recursos nativos do navegador

Console: objeto que representa o terminal interativo do navegador. Algumas funções: log, info, warm e error.

Window: objeto que representa a aba atual do navegador. Algumas funções: alert, confirm e prompt.

Document: objeto que representa a página html atual.

Comparação de dados

A comparação de dois dados retorna, no console, um resultado do tipo booleano (True ou False). Confira abaixo os tipos de operadores dessa natureza.

    Comparação de Igualde: == e ===

  1. == -> Compara dois valores tentando igualar os seus tipos. Por exemplo:
  2. 10 == 10 (True)
  3. 10 == "10" (True)
  4. === (mais restrito/mais utilizado) -> Compara dois valores, considerando o seu tipo. Por exemplo:
  5. 10 === 10 (True)
  6. 10 === "10" (False)

    7. Comparação de Diferença: != e !==

  7. != -> Compara dois valores tentando igualar os seus tipos. Por exemplo:
  8. 5 != 5 (False)
  9. 3 != 5 (True)
  10. 3 != "5" (True)
  11. !== (mais restrito/mais utilizado) -> Compara dois valores, considerando o seu tipo. Por exemplo:
  12. 5 != "5" (True)

    13. Maior/menor e Maior/Menor ou igual

  13. >, >=, <, <=

Operadores Lógicos

Servem para avaliar operações envolvendo números booleanos. Ou seja, expressões envolvendo valores verdadeiro e falso.

E (AND) - &&

Só é verdadeiro se ambas as partes forem verdadeiras.

true && true === true
true && false === false
false && true === false
false && false === false

Ou (OR) - ||

Só é falso se ambas as partes forem falsas.

true || true === true
true || false === true
false || true === true
false || false === false

Não (NOT) - !

Inverte o resultado (de verdadeiro para falso e vice-versa).

! true === false
! false === true

Como adicionar o JavaScript no documento HTML

Para incorporar ou referenciar um script, escrito em JavaScript, dentro de uma página web, utilizamos, no documento HTML, a tag <script>. Por exemplo:

Incluindo o código Javascript diretamente no HTML:

<script>
// Código JavaScript aqui
alert("Olá, mundo!");
<script>

Referenciando um arquivo externo que contém o código Javascript:

<script src="./meu-script.js"></script>

Observações:

A tag <script> pode ser utilizada tanto na parte <head> quanto na parte <body> de um documento HTML. No entanto, o posicionamento da tag <script> pode ter implicações no desempenho da página e na execução do código JavaScript. Quando você coloca a tag <script> no <head>, o script é baixado e executado antes do conteúdo do corpo (<body>) ser totalmente carregado. Isso significa que, se o script estiver manipulando elementos no DOM, pode haver problemas se esses elementos ainda não foram carregados. Por esse motivo, muitas vezes é recomendado colocar scripts no final do corpo (</body>) para garantir que o conteúdo HTML seja totalmente carregado antes que o script seja executado. Alternativamente, você pode usar o atributo defer na tag <script> para indicar que o script deve ser deferido até que o HTML seja completamente analisado.
Exemplo de colocação no final do corpo:
<!DOCTYPE html>
<html>
<head>
<!-- Meta tags, links para estilos, etc. -->
</head>
<body>
<!-- Conteúdo da página -->
<script src="./index.js"></script>
</body>
</html>
Exemplo de uso com defer no <head>:
<!DOCTYPE html>
<html>
<head>
<!-- Meta tags, links para estilos, etc. -->
<!-- O script será baixado de forma assíncrona, mas só será executado depois que o HTML for completamente analisado -->
<script defer src="./index.js"></script>
</head>
<body>
<!-- Conteúdo da página -->
</body>
</html>

Se você deseja exibir a tag <script> como texto na sua página HTML, mas não quer que o navegador a interprete como uma tag HTML, você pode usar entidades HTML para representar os caracteres especiais. No caso do caractere <, você pode usar & lt; (sem espaço), e para >, você pode usar & gt; (sem espaço). Isso fará com que o navegador exiba os caracteres < e > como texto simples, sem interpretá-los como tags HTML.
